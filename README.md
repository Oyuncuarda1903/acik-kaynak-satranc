# ♟️ Gelişmiş Satranç Oyunu

HTML, CSS ve JavaScript kullanarak geliştirdiğim, tarayıcı üzerinden oynanabilen açık kaynaklı bir satranç oyunudur.

## 🚀 Özellikler
* **Akıllı Analiz Çubuğu:** Sadece taş sayısına değil, taşların tahtadaki konumuna (merkez kontrolü, şah güvenliği vb.) göre anlık analiz yapar.
* **Düzeltilmiş Görseller:** Atlar ve diğer taşlar için yüksek kaliteli ve standart SVG tasarımları kullanıldı.
* **Süre Seçenekleri:** 5, 10 ve 20 dakikalık tempo seçenekleri ve aktif hamle sayacı.
* **Farklı Temalar:** Göz zevkine göre mor, ahşap veya klasik yeşil temalar arasında geçiş imkanı.

## 🛠️ Kullanılan Teknolojiler
* HTML5 & CSS3
* JavaScript (Vanilla JS)
* **Chess.js** (Satranç kuralları ve hamle kontrolü için)

## 🎮 Nasıl Oynanır?
1. Projeyi bilgisayarınıza indirin veya GitHub Pages linkine tıklayın.
2. `index.html` dosyasını tarayıcınızda açın.
3. Hamle yapmak istediğiniz taşa ve ardından gitmek istediğiniz kareye tıklayın.
